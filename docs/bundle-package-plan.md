## 1. 패키지 구성

### 패키지 A: 이사 기본 패키지
| 항목 | 내용 |
|------|------|
| 포함 서비스 | 이사 + 입주청소 |
| 타겟 | 이사 예정 가구주 (가장 일반적) |
| 할인율 | 청소비 10% 할인 |
| 가격 예시 | 이사 80만 + 청소 15만 → **청소 13.5만** (1.5만 절약) |
| CTA | "이사+청소 함께 신청하기" |

### 패키지 B: 이사 올인원 패키지
| 항목 | 내용 |
|------|------|
| 포함 서비스 | 이사 + 입주청소 + 인터넷 |
| 타겟 | 신규 입주자 (인터넷 신규 가입 필요) |
| 할인율 | 청소비 10% + 인터넷 지원금 추가 5만원 |
| 가격 예시 | 이사 80만 + 청소 13.5만 + 인터넷 지원금 48만→**53만** |
| CTA | "올인원 패키지 상담받기" |

### 패키지 C: 이사 프리미엄 패키지
| 항목 | 내용 |
|------|------|
| 포함 서비스 | 이사 + 입주청소 + 인터넷 + 에어컨 설치 |
| 타겟 | 신축 아파트 입주자 |
| 할인율 | 청소비 15% + 인터넷 지원금 +5만 + 에어컨 설치비 10% |
| 가격 예시 | 이사 80만 + 청소 12.75만 + 인터넷 53만 + 에어컨 8만→**7.2만** |
| CTA | "프리미엄 패키지 무료 상담" |

## 2. 수익 구조

### 번들 수수료 모델

| 서비스 | 단독 수수료 | 번들 수수료 | 차이 |
|--------|:---:|:---:|:---:|
| 이사 매칭 | 3~8만원/건 | 3~8만원 (동일) | — |
| 입주청소 | 2~5만원/건 | 2~5만원 (동일) | — |
| 인터넷 CPA | 8~15만원/건 | 8~15만원 (동일) | — |
| 에어컨 | 3~8만원/건 | 3~8만원 (동일) | — |
| **번들 보너스** | — | **+2만원/패키지** | 추가 수익 |

### 번들 효과
```
단독 신청 시:  이사만 → 수수료 5만원
번들 A 신청 시: 이사+청소 → 수수료 5만 + 3만 + 보너스 2만 = 10만원
번들 C 신청 시: 이사+청소+인터넷+에어컨 → 5만 + 3만 + 12만 + 5만 + 2만 = 27만원
```

**핵심**: 이사 1건에서 최대 **5.4배** 수수료 확보 (5만→27만)

## 3. 운영 플로우

### 고객 플로우

```
1. 메인홈 번들 배너 클릭
   ↓
2. 패키지 선택 모달
   ├── A: 이사+청소 (가장 인기)
   ├── B: 이사+청소+인터넷
   └── C: 이사+청소+인터넷+에어컨 (최대 혜택)
   ↓
3. 기본 정보 입력
   - 이름, 연락처
   - 출발지/도착지 (카카오 주소)
   - 이사 날짜
   - 평수
   ↓
4. 패키지 상담 신청 완료
   ↓
5. SMS로 안내
   "○○님의 [이사 올인원 패키지] 상담이 접수되었습니다.
    담당 상담사가 30분 내 연락드립니다."
   ↓
6. 상담사 연락
   - 이사: 매칭 업체 3곳 추천
   - 청소: 일정 확인 후 배정
   - 인터넷: 통신사별 지원금 안내
   - 에어컨: 설치 일정 확인
```

### 업체 플로우

```
1. 패키지 상담 접수 (Supabase)
   ↓
2. 서비스별 분리 배정
   - 이사 → 이사 업체 SMS 발송
   - 청소 → 청소 업체 SMS 발송
   - 인터넷 → 인터넷 상담사 배정
   - 에어컨 → 에어컨 업체 SMS 발송
   ↓
3. 각 업체 → 고객에게 직접 연락
   ↓
4. 완료 후 리뷰 요청 SMS 자동 발송
```

## 4. 마케팅 전략

### 배너 문구 (A/B 테스트)

| 버전 | 문구 | 타겟 |
|------|------|------|
| A | "이사할 때 청소까지 한번에! 최대 10% 할인" | 가격 민감 |
| B | "이사+청소+인터넷 올인원! 번거로움 제로" | 편의 중시 |
| C | "780만 고객이 선택한 패키지 상담" | 신뢰 중시 |
| D | "신축 입주? 이사+청소+인터넷+에어컨 한번에" | 신축 입주 |

### 노출 위치

| 위치 | 배너 유형 |
|------|---------|
| 메인홈 부가서비스 아래 | 가로 스크롤 카드 3개 (A/B/C 패키지) |
| 이사 견적 완료 후 | "청소도 함께 신청하시겠어요?" 업셀 모달 |
| 인터넷 상담 완료 후 | "이사도 예정이신가요?" 크로스셀 모달 |
| 이사 날짜 선택 시 | "이사일에 맞춰 청소+인터넷도 한번에!" 배너 |

### 업셀/크로스셀 타이밍

```
이사 신청 완료 → "입주청소도 함께?" (업셀 전환율 예상 30~40%)
청소 신청 완료 → "인터넷 가입하면 지원금 48만원!" (크로스셀)
인터넷 신청 완료 → "에어컨 설치도 필요하시면?" (크로스셀)
```

## 5. DB 구조

### bundle_consultations 테이블

```sql
CREATE TABLE bundle_consultations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  phone TEXT NOT NULL,
  package_type TEXT NOT NULL CHECK (package_type IN ('basic', 'allinone', 'premium')),
  from_address TEXT,
  to_address TEXT,
  move_date TEXT,
  size TEXT,
  -- 서비스별 상태
  moving_status TEXT DEFAULT 'pending',
  clean_status TEXT DEFAULT 'pending',
  internet_status TEXT DEFAULT 'pending',
  aircon_status TEXT DEFAULT 'pending',
  -- 메타
  total_discount INT DEFAULT 0,
  memo TEXT,
  status TEXT DEFAULT 'pending',
  created_at TIMESTAMPTZ DEFAULT now()
);
```

### API

| 메서드 | 경로 | 기능 |
|--------|------|------|
| POST | /api/bundle | 번들 상담 신청 저장 |
| GET | /api/bundle?status=pending | 어드민 번들 조회 |
| PATCH | /api/bundle/:id | 서비스별 상태 업데이트 |
