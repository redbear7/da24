1. AI 견적 요청 (자동 견적 추정)

숨고 참고 "AI 견적 요청" 버튼 → 짐량/주소 기반 자동 견적

항목구현 내용
위치메인홈 상단 + 이사 모달 내
동작출발지/도착지 + 평수 + 짐량 입력 → 예상 견적 금액 자동 산출
로직평수별 기본 가격 + 거리 가중치 + 짐량 가중치
UI"예상 견적: 약 85만원 ~ 120만원" 범위 표시
파일lib/estimate-calculator.ts, components/EstimateBanner.tsx

예상 가격 로직

기본가 = 평수별 기준가 (10평 40만, 20평 70만, 30평 100만, 40평+ 130만)
거리 = 시내 0%, 시외 +10%, 장거리 +20%
짐량 = 기본 0%, 많음 +15%, 매우많음 +30%
예상가 = 기본가 × (1 + 거리 + 짐량) ± 15%

2. 업체 프로필 카드

숨고 참고 별점 5.0 + 리뷰수(68) + 경력 4년 + 포트폴리오

항목구현 내용
