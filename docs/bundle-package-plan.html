2. 수익 구조

단독 신청: 이사만 → 수수료 5만원
기본 패키지: 이사+청소 → 5만 + 3만 + 보너스 2만 = 10만원
올인원: 이사+청소+인터넷 → 5만 + 3만 + 12만 + 2만 = 22만원
프리미엄: 전체 → 5만 + 3만 + 12만 + 5만 + 2만 = 27만원

핵심: 이사 1건에서 최대 5.4배 수수료 확보 (5만 → 27만)
