# 숨고 스타일 기능 적용 기획서

**작성일**: 2026-04-14
**참고**: soomgo.com 이사/청소 카테고리

## 적용할 기능 5가지

### 1. AI 견적 요청 (자동 견적 추정)

**숨고 방식**: "AI 견적 요청" 버튼 → 짐량/주소 기반 자동 견적
**da24 적용**:

| 항목 | 구현 |
|------|------|
| 위치 | 메인홈 상단 + 이사 모달 내 |
| 동작 | 출발지/도착지 + 평수 + 짐량 입력 → 예상 견적 금액 자동 산출 |
| 로직 | 평수별 기본 가격 + 거리 가중치 + 짐량 가중치 |
| UI | "예상 견적: 약 85만원 ~ 120만원" 범위 표시 |
| 파일 | `lib/estimate-calculator.ts`, `components/EstimateBanner.tsx` |

**예상 가격 로직 (초안)**:
```
기본가 = 평수별 기준가 (10평 40만, 20평 70만, 30평 100만, 40평+ 130만)
거리 = 시내 0%, 시외 +10%, 장거리 +20%
짐량 = 기본 0%, 많음 +15%, 매우많음 +30%
예상가 = 기본가 × (1 + 거리 + 짐량) ± 15%
```

### 2. 업체 프로필 카드 (고수 카드 스타일)

**숨고 방식**: 별점 5.0 + 리뷰수(68) + 경력 4년 + 포트폴리오 사진
**da24 적용**:

| 항목 | 구현 |
|------|------|
| 위치 | /moving 업체 리스트, 메인홈 추천 업체 |
| 카드 구성 | 업체명 + ★ 평점(리뷰수) + 경력 + 전문 분야 태그 + 한줄 소개 |
| DB | moving_companies 테이블에 `career_years`, `portfolio_url` 컬럼 추가 |
| 뱃지 | premium/vip 업체에 "추천" 뱃지 |
| 파일 | `components/CompanyCard.tsx` |

**카드 디자인**:
```
┌──────────────────────────────┐
│  🏠 창원 해피무브              │
│  ★ 4.9 (412) · 경력 8년       │
│  가정이사 · 포장이사 · 에어컨    │
│  "이사+에어컨 원스톱 서비스"     │
│  [견적 요청하기]               │
└──────────────────────────────┘
```

---

### 3. "오늘의 추천 업체" 섹션 (광고 모델)

**숨고 방식**: "오늘의 추천 고수" + 카테고리 필터 탭 + 광고 표시
**da24 적용**:

| 항목 | 구현 |
|------|------|
| 위치 | 메인홈 고객 평가 아래 |
| 구조 | 카테고리 탭 (이사/청소/인터넷/에어컨) + 업체 카드 가로 스크롤 |
| 정렬 | tier(vip→premium→basic) + 평점 순 |
| 광고 | premium/vip 업체에 "광고" 태그 표시 |
| DB | moving_companies.tier 기반 |
| 파일 | `components/RecommendedCompanies.tsx` |

---

### 4. 서비스 검색 + 지역 자동 감지

**숨고 방식**: "창원시 의창구 ∨ 에서 어떤 서비스가 필요하세요?"
**da24 적용**:

| 항목 | 구현 |
|------|------|
| 위치 | 메인홈 상단 (이사 카테고리 위) |
| 동작 | GPS 자동 감지 → "창원시 의창구" 표시 + 서비스 검색 입력 |
| 검색 | "이사", "청소", "인터넷", "에어컨" 키워드로 해당 페이지 이동 |
| 파일 | `components/ServiceSearch.tsx` |

**UI**:
```
┌─ 📍 창원시 의창구 ∨ ─────────────────────┐
│  🔍 어떤 서비스가 필요하세요?              │
└──────────────────────────────────────────┘
```

---

### 5. 서비스 번들 배너 ("한번에 견적 받기")

**숨고 방식**: "예산을 아끼는 오프라인 창업" 같은 테마 번들 카드
**da24 적용**:

| 항목 | 구현 |
|------|------|
| 위치 | 메인홈 부가서비스 아래 |
| 번들 | 이사+청소 패키지, 이사+인터넷+에어컨 올인원 |
| UI | 카드형 배너, 가로 스크롤 |
| 동작 | 클릭 → 번들 상담 신청 모달 |

**번들 예시**:
```
이사 올인원 패키지
이사 + 입주청소 + 인터넷 + 에어컨
한번에 신청하고 최대 30% 절약!
[패키지 상담 신청하기]
```
